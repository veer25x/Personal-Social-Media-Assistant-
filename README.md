# Personal-Social-Media-Assistant-(https://personal-social-media-assistant.onrender.com)
AI-powered social media assistant that generates posts, schedules content, tracks analytics, and creates smart replies - all in one dashboard.

Personal Social Media Assistant is an AI-powered web application that helps users manage and grow their social media presence automatically. It generates engaging content, schedules posts, tracks analytics, and creates smart replies - all from one dashboard.

✨ Key Features
AI Content Generator - Creates posts for Twitter, LinkedIn, and Instagram with customizable tones (casual, professional, viral, motivational)

Smart Hashtags - Generates optimized, trending hashtags for better reach

Post Scheduler - Schedule posts for future publishing with automatic posting simulation

Analytics Dashboard - Track engagement metrics, views, and get AI-powered insights

Reply Generator - Creates contextual replies to comments in friendly, witty, or professional tones

Content Ideas - Generates 10 viral post ideas based on your niche

🛠️ Tech Stack
Backend: Python + Flask

Frontend: HTML + Tailwind CSS

Database: SQLite

AI: OpenAI API (with mock mode fallback)

🎯 Use Cases
Social media managers

Small business owners

Content creators

Digital marketers

Personal brands

💡 Value Proposition
Save hours of manual work by automating content creation, scheduling, and engagement. Get data-driven insights to improve your social media strategy.
